Voici un site que j'ai construit qui résume mes compétences acquise durant le stage et qui contient le lien pour accéder à mon rapport de stage : lien.

Compétences Techniques (Savoir-faire) :

  • Design et mise en page : Personnalisation graphique en CSS (gestion des dégradés, des polices, adaptation des tailles d'images).

  • Logique de programmation : Création d'un script dynamique en JavaScript pour l'interactivité (gestion des événements avec le clic de souris).

    • Compétences Professionnelles (Savoir-être) :

    • Autonomie et recherche : Capacité à chercher des solutions face à un bug technique (comme le nettoyage des URL d'images).

    • Écoute et application : Intégration des "Astuces DSI" données par les professionnels de Covage Infra.

    Les plus grandes difficultés que j'ai rencontrées en codant ce site :

    • Apprendre à nettoyer les URL d'images provenant des moteurs de recherche pour pouvoir les intégrer au site ? Astuce DSI : Il faut extraire l'URL source directe de l'image (qui finit souvent par .png ou .jpg) et supprimer tout le reste du lien.
    • Apprendre à gérer la taille d'une image sans la déformer ? Astuce DSI : En CSS, on fixe une largeur (ex: width: 300px) et on applique la propriété height: auto pour garder les proportions.

    Les différents langages utilisés pour coder un site :

    • Le HTML : C'est ce qui permet d'écrire dans le site, il constitue la base et la structure.
    • Le CSS : Il sert à modifier l'apparence et le design du site.
    • Le JavaScript : Il sert d'ordinaire à rendre le site dynamique et interactif.
